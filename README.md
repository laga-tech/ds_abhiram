# Trader Behavior vs Market Sentiment Analysis

## Overview
This project analyzes how trader behavior varies under different market sentiment conditions (Fear and Greed) using historical trading data and sentiment indicators.

## Objectives
- Study profitability patterns under Fear vs Greed
- Analyze risk exposure and trade volume
- Identify actionable trading insights

## Methodology
- Data cleaning and preprocessing
- Feature engineering for profitability and risk
- Sentiment-based aggregation
- Exploratory data analysis with visualizations

## Project Structure
- notebook_1.ipynb : Complete analysis (Google Colab)
- csv_files/ : Final processed datasets
- outputs/ : Visualizations generated from analysis
- ds_report.pdf : Final analytical report
